AI Voice Assistant

Powered by ElevenLabs and Zapier Integration

ElevenLabs Voice Assistants

Create advanced AI voice assistants using ElevenLabs' lifelike voices. These assistants can answer questions, have conversations, and provide helpful information.

Features include:

  • Lifelike text-to-speech
  • Multiple voice options
  • Knowledge base integration

Zapier Integration

Connect your voice assistants to thousands of apps and services through Zapier. Automate tasks and extend your assistant's capabilities.

Integrations include:

  • Gmail for email communication
  • Google Calendar for scheduling
  • Google Sheets for data tracking
  • And many more services

Customizable Configuration

Easily configure your assistants to fit your specific needs. Set up Zapier endpoints, choose voice options, and customize assistant behavior.

Configuration options:

  • Custom Zapier MCP endpoint
  • Voice selection and parameters
  • System prompts and instructions
  • Knowledge base content

Zapier MCP Configuration

Enter your Zapier MCP endpoint to configure the integration. This allows your voice assistants to connect with Zapier's services.

Your current endpoint configuration is stored in your browser's local storage.

Using in Cursor IDE

This application is designed to be run in Cursor IDE with ElevenLabs and Zapier MCPs enabled. To run the demo:

  1. Clone the repository
  2. Open in Cursor IDE
  3. Make sure ElevenLabs and Zapier MCPs are enabled
  4. Run src/runMcpDemo.js

Example code to create a voice assistant:

const agent = await mcp_ElevenLabs_create_agent({
  name: "Customer Service Assistant",
  first_message: "Hello! How can I help you today?",
  system_prompt: "You are a helpful customer service AI assistant.",
  voice_id: "cgSgspJ2msm6clMCkdW9", // Adam voice
  language: "en",
  temperature: 0.7
});
